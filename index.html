How to Use This Reference

Prefixes change the direction or meaning of roots (e.g., sub- means "under").

Suffixes determine word class or add meaning (e.g., -ment indicates "result of action").

Roots provide core meaning (e.g., bio means "life").

Combine these elements to decode complex terms like photosynthesis = photo- (light) + syn- (together) + thesis (placing).
